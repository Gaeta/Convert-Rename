# Convert Rename

## Install

```powershell
npm install
node . [args]
```

#### Example

```powershell
npm install
node . --quality 45
```

## Command line options

*output*
 Type: `folder`
 Default: `./output`

*input*
 Type: `folder`
 Default: `./input`

*rename*
 Type: `boolean | string | uuid string`
 Default: `uuid` _uuid will give random uuid_

*compress*
 Type: `boolean`
 Default: `true`

*quality*
 Type: `number`
 Default: `70`
